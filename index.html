Welcome to my bash scripts collection.

This a collection of my bash scripts. Some of them are written for personal use, so they're adapted to my needs, feel free to modify them.

backup.sh

It was my backup script for a while. Quite simple, just some rsync over different folders, with logs. But in my case I had first to mount my device.

getip.sh

Obtain ips from given domains. With google reader support. Three modes of execution:

  • 1. Single url. Type a single url and return its ip
  • 2. Url file. Several urls stored in a file (one per line)
  • 3. OPML File (Exported from Google Reader). Maybe the most interesting. Type the path of the .opml file, and it will generate a file with all ips. This may take a while

remote_access.sh

There's two version of this script, to fill my needs. Using nmap, it scans all your network looking for online hosts. Syncs some folders and then send a shutdown command.

reset-nic.sh

During this course, in class we created a lot of VMs, which gave us a lot of problems with network cards, because we had a "clean" machine, and then we make clones and clones, so NICs problems arised eventually. Duplicated NICs, too many eth's, etc etc. With this script you may solve those problems. It removes your 70-persistent-net.rules and unload / load each network driver, which causes a regeneration of that file with the correct configuration
